Interactive Learning Platform 

The Interactive Learning Platform is the main module of the project. It is designed to provide a personalized and stress-aware learning experience. Instead of using a one-size-fits-all approach, the system interacts with the user, understands their inputs, and suggests suitable learning paths, activities, or strategies.

The platform begins with user authentication. After login or registration, the user is taken to the dashboard, which serves as the central hub of the system. From the dashboard, the user can access learning recommendations, mental wellness activities, and other interactive features.

The system observes the user’s behavior, responses, and activity patterns. Based on this information, it suggests different learning paths or stress-relief activities. For example, if the user shows signs of stress or low engagement, the platform may recommend a short breathing exercise or a relaxation activity before continuing with learning tasks.

This creates a balanced environment where the system supports both academic progress and mental well-being.

Key Functional Points

1. User Authentication
Login and registration system
Stores user data locally for personalization

2. Dashboard Interface
Central hub for all features
Shows learning suggestions and wellness options

3. Interactive Learning Guidance
Takes user inputs and preferences
Suggests learning paths or strategies

4. Stress-Aware System
Monitors user activity and responses
Detects signs of stress or fatigue

5. Wellness Integration
Suggests relaxation activities
Includes breathing exercises, meditation, and mini-games

6. User Settings
Theme selection (light/dark mode)
Language preferences
